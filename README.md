# ⚛️ Minha Primeira Página em React com TypeScript

Bem-vindo ao repositório da minha primeira página web construída utilizando **React**! Este projeto marca o início da minha jornada com a biblioteca, focando fortemente no aprendizado de componentização, organização arquitetural de pastas e tipagem estática com **TypeScript**.

O objetivo principal foi criar uma página inicial (Landing Page) modular, onde cada seção da tela foi desenvolvida de forma isolada e inteligente.

---

## 📁 Estrutura de Pastas e Arquivos

```text
src/
├── assets/
│   └── reactLogo.png       # Recursos estáticos e imagens do projeto
├── components/             # Blocos reutilizáveis de interface (UI)
│   ├── cards.tsx           # Componente de cards de exibição
│   ├── footer.tsx          # Rodapé da página
│   ├── header.tsx          # Cabeçalho
│   └── hero.tsx            # Seção principal/banner de destaque (Hero Section)
├── pages/                  # Telas ou páginas completas da aplicação
│   └── home.tsx            # Página inicial que agrupa todos os componentes
├── styles/                 # Estilização visual (CSS tradicional)
│   ├── cards.css           # Estilo exclusivo dos cards
│   ├── footer.css          # Estilo exclusivo do rodapé
│   ├── global.css          # Regras globais, fontes e resets do sistema
│   ├── header.css          # Estilo exclusivo do cabeçalho
│   └── hero.css            # Estilo exclusivo do banner principal
├── App.tsx                 # Componente raiz que inicializa a estrutura
└── main.tsx                # Ponto de entrada que renderiza o React no DOM do navegador

```
**Este projeto representa o ponto de partida dos meus estudos em React, materializando o conhecimento adquirido nesse primeiro contato. Daqui para frente, meu objetivo é aperfeiçoar minhas habilidades, conhecer novas técnicas e evoluir na criação de sistemas.**
